# Project: Ahola Apparel Co.
## About this project
You can see this project on-line, going to [Alohaapparel On-Line](http://margaomar.github.io/Aloha/index.html)

## Goals of the site
This site has been created to learn how to create a web site using HTML5, CSS3, JQUERY and GIT. 

> Hand-Crafter Styles
Aloha Apparel Co. sells only the finest ready-to-travel clothing. Browse our current selectin of products or sign-up for updates below.
